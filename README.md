NAVI CARE - UI
It is the interface work of the food ordering application. It consists of twelve pages in total. These pages:

- Splash Screen
- Sign in 
- Sign up
- Authentication Page
- Home Screen
- Alert Page
- Hospital Page
- Hospital Info Page
- Doctors Page
- Doctors Info Page
- Search Page
- Profile Page
